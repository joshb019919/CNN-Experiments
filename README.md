# CNN-Experiments

Perform six experiments on convolutional neural nets (CNNs) in varying optimizer, model, and number of convolutional layers.

## Usage

For Windows users:
    In the setup folder, double-click setup_env.bat.
    In the setup folder, double-click run_all_experiments.bat.

For Linux users:
    In the terminal, do `./setup/setup_env.sh`.
    In the terminal, do `./setup/run_all_experiments.sh`.
